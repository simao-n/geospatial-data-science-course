### 2. Run Locally

If you prefer to run notebooks on your machine:

1. Install [Python](https://www.python.org/) and [Jupyter](https://jupyter.org/install)
2. Clone the repository:
   ```bash
   git clone https://github.com/tamagusko/geospatial-data-science-course.git
   cd geospatial-data-science-course
   ```
3. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Start Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
5. Open the relevant notebook from the `notebooks/` folder.
